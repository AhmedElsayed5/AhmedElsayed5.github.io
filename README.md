# Around The U.S.

### Description

- Social Media Platform where users get the chance to share their pictures from places around the US
- Responsive Design
- Maintains Privacy for users
- Aesthetic Design
- Animated Transition
  
**Technologies**

- Javascript
- HTML
- CSS
- WebPack
